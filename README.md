# 未下完的棋 · 试玩版

一个基于围棋规则的 Roguelike 策略计分小游戏的试玩 Demo。执黑落子、吃白触发斜对角染色、终局围眼计分。

🎮 **在线试玩**:<https://yizhengyuan.github.io/wxw-go/>

## 玩法

- 你执黑先行,电脑执白。
- 黑棋吃掉白子时,被吃点四周的**方格(格子,非交叉点)**永久 +1 分数价值(橙色数字显示在格子里)。
- **终局**:双方棋子用尽且无处可下后,被黑棋正交包围的空交叉点即为「眼」。
- **计分**:单眼 = 1(基础) + 该眼四周方格的染色值之和(中腹眼 4 格 / 边眼 2 格 / 角眼 1 格)。总分 ≥ 目标分即过关。

## 本地运行

纯原生 HTML / CSS / JavaScript,无需构建。直接用浏览器打开 `index.html` 即可,或起一个本地静态服务器:

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 目录结构

```
index.html        # 入口页面
style.css         # 样式
js/
  config.js       # 配置(棋盘尺寸、棋子数、目标分等)
  board.js        # 棋盘与基础围棋规则(气、提子、禁着)
  scoring.js      # 染色与终局围眼计分
  ai.js           # 白棋 AI
  game.js         # 对局流程控制
  render.js       # Canvas 渲染
  main.js         # 入口装配
```
